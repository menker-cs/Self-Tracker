# Self-Tracker

## ℹ️Instructions
* **Step 1**: Download the DLL and put it in your plugins folder
* **Step 2**: Open the game then close out of it once it fully loads
* **Step 3**: Navigate to your `Gorilla Tag` folder then locate the folder named `SelfTracker`
* **Step 4**: Replace the text in `webhook.txt` with your webhook url
* **Step 5**: Replace the text in `name.txt` with your name
